## How to Install Livebook

To install Elixir's Livebook (given Elixir 1.14+ installation):

```sh
mix do local.rebar --force, local.hex --force
mix escript.install hex livebook
# if using asdf:
# asdf reshim elixir
```

Running:

```sh
livebook server
```
